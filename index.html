The Front-End

To understand what front-end is, you need to know what client-server architecture is. This model assumes two agents: the so-called client-side involves languages that are processed by the user's browser, the server-side refers to what is processed on the server and makes use of different languages. Although different, the two sides need to work together to ensure the usability of the site and a good navigation experience. In this context, front-end programming is associated with the client-side. The front end is where we find the "expensive" of a website or application, with design, navigation interface and user interaction tools, search areas and forms.

What's it for?

The front-end developer is responsible for the user experience within a web application, it is he who will design and develop the pages with which the user will later interact.
